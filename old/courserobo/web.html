Web Design (12-16 years old)


No.
Lesson Topic


Key Objective

1 UI vs. UX vs. Web Design
Student will know:
- Know the different between UI design / UX design and Web design.
- Give the example of UI design / UX design and Web design.

2 Picking the right tool
Student will know:
- Choose the right tool for create their website.
- Know the different between web create engine.
3 Color Theory
Student will know:
- Know how to use color for website.
- Give the example of type of color.

4 Color Harmony
Student will know:
- Create a color palette for projects.

5 Typography
Student will know:
- Choose the best typography for create their website.
- Know create the typography for projects.

6 Spacing and Sizing
Student will know:
- Know how to use spacing and sizing theory in their website.
- Give the example of spacing and sizing theory.
7 Icon and Image
Student will know:
- Know how to use icon and image for web design.
- Create icon and image for projects.

8 Header
Student will know:
- Know definition and use case for header.
- Design a web header.

9 Buttons
Student will know:
- Know how to use button in web design.
- Design a button.
10 Mini Projects
Student will know:
- Create mini website.
